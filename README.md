# Apurv Gupta | Portfolio

A high-performance, cinematic developer portfolio. This project serves as a digital signature, showcasing my expertise across several domains

## 🛠 Tech Stack

### Frontend & Animation

- **Framework:** [React 19](https://react.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/) (Strict Mode)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)

---

### Animation Engines:

- **Framer Motion:** Used for layout transitions, scroll-linked parallax, and skill-card interactions.
- **GSAP (GreenSock):** Powers the high-precision "Text Type" typewriter effects and cinematic cursor logic.
- **Video Processing:** Optimized WebM/MP4 pipeline for the cinematic "Curtain" preloader.

---

### Backend & Infrastructure

- **Deployment:** [Vercel](https://vercel.com/)
- **CI/CD:** GitHub Actions

---

### ✨ Cinematic Curtain Preloader

A bespoke loading sequence featuring a hardware-accelerated video mask.

- **Cinematic Exit:** A standard fade, the intro slides horizontally to the right using a `cubic-bezier` Quart transition.
- **Smart Loading:** Real-time progress tracking of video buffer before lifting the curtain.

---

## 🏛 Architectural Principles

- **Entropy Resistance:** Code structured to remain maintainable as features scale.
- **Visual Weight:** Using motion to guide the user's eye toward "Selected Projects" and "Professional Focus."
- **Efficiency:** Minimal re-renders through memoized animation variants and optimized `IntersectionObservers`.

## Getting Started

1. **Clone the repository:**

```bash
git clone https://github.com/Apurv7Gupta/Apurv-Gupta-Portfolio_Prototype.git

```

2. **Install dependencies:**

```bash
npm install

```

3. **Run the development server:**

```bash
npm run dev

```

4. **Build for production:**

```bash
npm run build

```
